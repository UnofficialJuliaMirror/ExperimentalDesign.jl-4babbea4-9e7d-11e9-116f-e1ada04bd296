ExperimentalDesign.variance_prediction! — Method.
variance_prediction!(dispersion_matrix::Array{Float64,2}, model_matrix::Array{Float64,2}, candidate_set::Array{Float64,2}, design_variance::Array{Float64,1}, candidate_variance::Array{Float64,1}, predictions::Array{Float64,2}, deltas::Array{Float64,2})

Computes deltas for the exchanges between all pairs made between experiments in model_matrix and experiments in candidate_set. The dispersion_matrix array must contain the dispersion matrix for model_matrix. Both model_matrix and candidate_set must be model matrices. The arrays design_variance, candidate_variance, predictions and deltas will be overwritten.
